Marca y modelo Se nos olvidó apuntarlo, pero en otra práctica que hicimos había un Maxtor DiamondMax. Como ejemplo, aquí tenemos uno de esta marca, el modelo 6B200P0.

Interfaz (IDE/SATA) IDE, y para configurar si será maestro o esclavo, habrá que puentear las conexiones de acuerdo con las instrucciones.
